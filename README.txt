# activite_OC
Activité OpenClassrooms pour le cours sur utilisation de git

Bonjour,

Voici le code pour un petit programme en java qui simule l'entrée d'un tronc d'arbre (classe Buche) dans une scierie et calcule le nombre en lequel ce tronc a été débité et l'affiche.

Les dimensions des troncs sont rentrées en dur dans les paramètres du constructeur Buche présent dans la méthode main de la classe Scierie.

En espèrant avoir répondu à l'exercice.
